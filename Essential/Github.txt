For contribution processs -----------------------------------
  - Go to main repo > click on Fork > create fork // it will be copy in your github acc.
  - Clone fork repo code from your github acc
      git clone repo link
      cd project name
      code .
  - Check and review code.
  - create a branch an give branch name related to your contribute type or feature.
      git checkout -b blog
  - continue your work.
  - git add .
  - git commit -am 'commit'// use when we modify code in existing file.
  - git push origin branch name
  - For merge repo - goto repository - click > compare & pull request > add title > 
      add description > create pull request.
  - Goto main auther github account > goto repo > click Merge pull request > confirm merge.
  - Goto your github account > goto repo > Sync fork > update brunch.
