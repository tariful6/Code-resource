Tanstack query is a data-fetching library.
if we use it. we have no need to use useeffect handleing.

1. npm i @tanstack/react-query

2. insert this in main.jsx component file :
	import { QueryClient,QueryClientProvider,useQuery} from '@tanstack/react-query'
	const queryClient = new QueryClient()

3. wrap HelmetProvider using - QueryClientProvider

        <AuthProvider>
            <QueryClientProvider client={queryClient}>
               <HelmetProvider>
                  <RouterProvider router={myRouter} />
               </HelmetProvider>
            </QueryClientProvider>
         </AuthProvider>
